Menggunakan nested-property, kita bisa menghindari penulisan ulang di CSS beberapa kali. Misalnya, menggunakan font sebagai namespace, yang menggunakan beberapa properti seperti font-family, font-size, dll. Dalam CSS biasanya kita harus menuliskan properti ini setiap kali dengan namespace. Menggunakan SASS, Kita bisa membuat sarang properti dengan menulis namespace hanya sekali.
